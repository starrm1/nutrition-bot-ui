# nutrition-bot-ui

UI Design Previews for the Nutrition Bot Management System.

---

## 🎨 Design Options

Four distinct visual design options are provided as standalone HTML mockups. Open each file directly in a browser to preview.

| File | Style | Theme | Key Traits |
|------|-------|-------|------------|
| [minimalist-design.html](minimalist-design.html) | **Option A – Minimalist/Clean** | Light | Whitespace-focused, single green accent, flat design |
| [dashboard-design.html](dashboard-design.html) | **Option B – Dashboard-Heavy** | Dark | Data-dense, charts, GitHub-style dark theme, monospace metrics |
| [card-based-design.html](card-based-design.html) | **Option C – Card-Based/Modular** | Dark + vibrant | Icon-heavy cards, drag-and-drop-ready, purple/teal gradients |
| [clinical-design.html](clinical-design.html) | **Option D – Professional/Clinical** | Light | Healthcare-inspired, WCAG AA accessible, blues & greens |

---

## 📋 Each Preview Includes

- Header / Navigation bar
- Bot Status Display (19 sample bots)
- Main Dashboard / metrics area
- Case Management panel
- Report Generation section
- Control Panel with toggles
- Sidebar / Menu
- Color palette documentation
- Typography specifications

---

## 🖌️ Design Specifications

### Option A – Minimalist/Clean
- **Colors:** White · Light gray `#F5F5F5` · Accent green `#4CAF82`
- **Typography:** Inter · Weight 400/500/600
- **Style:** Flat, spacious, content-focused

### Option B – Dashboard-Heavy (Dark Mode)
- **Colors:** BG `#0D1117` · Surface `#161B22` · Accent blue `#58A6FF`
- **Typography:** System sans + Courier New for metrics
- **Style:** Dense data grid, chart-forward, professional

### Option C – Card-Based/Modular
- **Colors:** BG `#1A1D2E` · Card `#2D3154` · Accent purple `#7C6AF7` · Teal `#22D3CC`
- **Typography:** Nunito · Bold 700/800 for values
- **Style:** Modular cards, icon-heavy, drag-and-drop-ready

### Option D – Professional/Clinical
- **Colors:** BG `#F0F4F8` · Clinical blue `#1A6FA8` · Medical green `#2E8B6E`
- **Typography:** Segoe UI · Structured hierarchy
- **Style:** Healthcare-grade, WCAG AA compliant, trust-focused
